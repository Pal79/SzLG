# Szórend (word order)

- Kijelentő mondatban:
   - határozószó + alany + állítmány + tárgy + határozók.
- Kérdőszóval ellátott kérdőmondat:
   - kérdőszó + segédige + alany + főige + határozók/egyéb
- Eldöntendő kérdés:
   - segédige + alany + föige + egyéb (safe szabály)
