# Elektromos áram
A töltéshordozók (elektornok vagy ionok) rendezett egyirányú mozgása.
|     |     | Az elektromos áram felosztása |     |     |
| :-: | :-: |             :-:               | :-: | :-: |
| **Szállítási áram** |  |  |  | **Vezetési áram** |
| Amikor a töltéshordozók a vezető anyagával együtt végeznek rendezett egyirányú mozgást (pl.: elhajított kulcscsomó) |  |  |  | Amikor a töltéshordozók a vezetőben uralkodó térerősség (feszültség) hatására végeznek egyirányú egyenletes mozgást |
|    |     |                                | **Váltakozó áram** | **Egyenáram** |

```mermaid
    A[Az elektromos áram felosztása] -->
    B --> [Szállítási áram]
    B --> [Vezetési áram]
```
